# MimoBOOK
Proyecto final de Desarrollo Movil


# Problemática:
mucha gente guarda sus notas, horario/agenda o información que usa día a día en distintas aplicaciones y es posible que no vuelvan a ver eso que necesitan otra vez.

# Propuesta:
mi propuesta de aplicación es hacer una aplicación que reúna las características de una agenda, post de notas, una alarma y un calendario en tu teléfono para mayor comodidad y eficiencia.
